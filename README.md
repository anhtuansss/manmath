# ManMath

ManMath is a web app for practicing Vietnamese high school math exams.

## Tech Stack

- Frontend: Next.js App Router + TypeScript + Tailwind CSS
- Backend: Express + TypeScript
- Database: PostgreSQL

## MVP Features

- Exam list
- Exam detail
- Timer
- Answer selection
- Submit exam
- Score calculation
- Review answers

## How to Run

## Frontend

```bash
cd frontend
npm install
npm run dev
```

## Backend

```bash
cd backend
npm install
npm run dev
```
