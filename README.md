# ManMath

## Tech stack
- Frontend: Next.js
- Backend: Node.js (Express)
- Database: MongoDB

## Features
- OCR bài toán
- Chấm điểm tự động

## Run project
1. Thêm URI MongoDB vào file `.env`
2. Chạy Backend: `cd backend && npm install && npm start`
3. Chạy Frontend: `cd frontend && npm install && npm run dev`
